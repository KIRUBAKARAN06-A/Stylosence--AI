# Stylosence--AI
AI-powered phygital retail system with recommendation, virtual try-on, and smart billing.


## Project Overview

Stylosense-AI is an AI-powered smart retail system that provides customers with personalized dress recommendations based on body measurements, height, weight, and shopping preferences.

### Features
- Body Scanning
- AI Dress Recommendation
- Virtual Try-On
- Smart Navigation
- Automated Billing
- Theft Detection
